# Tour Gangwon App

강원도 여행 추천 앱입니다.

## 설치 및 실행

```bash
# 의존성 설치
flutter pub get

# 앱 실행
flutter run
```

## 주요 기능

- 여행지 추천
- 즐겨찾기 관리
- 날짜별 추천
- 사용자 인증
